# HTML / PPT 幻灯片制作

当用户要做 PPT、路演材料、项目汇报、技术分享、课程课件，或想把 PPT 改造成更好的网页演示时，使用本技能。

## 输出目标

- 默认生成一个自包含 HTML 文件，内联 CSS 和 JS。
- 每页必须完整适配一个视口，不允许页内滚动。
- 视觉方向要明确，避免默认紫色渐变、系统字体和模板感。
- 优先使用强排版、几何图形、渐变、网格、噪声、动效，而不是堆图片。

## 标准流程

1. 确认用途：路演、教学、内部汇报、项目复盘、技术分享。
2. 整理内容：标题、核心观点、每页要完成的任务。
3. 选择风格：大胆发布感、科技暗色、纸张编辑感、瑞士现代、复古杂志等。
4. 生成幻灯片：语义化 `section`，键盘/触摸/滚轮导航，进度提示。
5. 检查适配：1920x1080、1280x720、768x1024、375x667。

## 页面密度限制

- 标题页：1 个标题 + 1 个副标题。
- 内容页：1 个标题 + 4 到 6 个要点。
- 卡片页：最多 6 张卡片。
- 代码页：最多 8 到 10 行。
- 图片页：1 张主图，限制高度。

## 必备 CSS 约束

- `.slide` 使用 `height: 100vh; height: 100dvh; overflow: hidden;`
- 字号、间距使用 `clamp()`。
- 图片必须有 `max-height`。
- 内容放不下就拆页，不要把字缩到不可读。
- 支持 `prefers-reduced-motion`。

## 交付说明

交付时说明：

- 文件路径。
- 页数。
- 使用的视觉风格。
- 可快速改的主题变量。
- 如何本地打开，例如 `xdg-open presentation.html`。
