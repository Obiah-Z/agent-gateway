# 工具使用说明

## 可用工具

你当前可以使用的工具会随着运行阶段不同而变化，常见工具包括：

### 文件工具
- **read_file**：读取工作区内文件内容
- **write_file**：写入文件内容，必要时自动创建父目录
- **edit_file**：替换文件中的精确文本片段（`old_string` 必须唯一）
- **list_directory**：列出目录中的文件和子目录

### 记忆工具
- **memory_write**：保存值得长期保留的信息
- **memory_search**：按关键词检索既有记忆

### 联网搜索工具
- **web_search**：使用 Tavily 搜索公网信息，适合查询最新信息、外部事实、政策、价格、版本、公司/人物动态等可能变化的问题
- **fetch_url**：通过 Tavily 提取指定 URL 的正文内容，用于核验搜索结果中的关键来源

### 系统工具
- **bash**：执行 shell 命令，但需要遵守安全限制
- **get_current_time**：获取当前日期和时间

## 使用原则

- 修改文件前，优先先读取文件内容
- 当用户提供偏好、事实、决定或背景时，主动使用 `memory_write`
- 在回答和历史上下文相关的问题前，可以先用 `memory_search`
- 遇到当前日期之后可能变化的问题，必须先用 `web_search` 检索
- 搜索后回答要尽量附来源 URL；关键结论应优先用 `fetch_url` 打开来源核验
- 如果联网搜索失败、来源冲突或证据不足，明确说明限制，不要编造
- 如果搜索结果需要被其他 Agent 后续使用，用 `memory_write` 保存摘要、来源 URL 和检索日期
- 控制工具输出长度，避免无关信息占满上下文
