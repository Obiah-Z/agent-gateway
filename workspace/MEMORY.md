# 长期记忆

> 这个文件用于记录那些不会每天变化、但长期有价值的事实和偏好。
> 智能体会在每次会话启动时读取这里的内容，用作基础上下文。

## 用户偏好

- 更偏好简洁回答，不喜欢冗长解释
- 时区：Asia/Shanghai（UTC+8）
- 主要使用中文，同时可以接受英文技术术语
- 需要推进项目时，偏好先规划、再分阶段实现、测试、提交和推送

## 重要背景

- 当前正在做 AI Agent / Gateway 相关项目
- 对系统架构和设计模式有持续兴趣
- 重视代码质量和清晰的抽象边界

## 记忆边界

- 长期记忆只记录稳定偏好、长期项目背景和明确决策。
- 不记录一次性 Cron 输出、临时错误、工具日志、普通搜索结果或短期调试过程。
- 如果信息可能很快过期，应放在会话回复或运行日志中，而不是写入长期记忆。
