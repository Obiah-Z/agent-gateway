# 工具使用

- `internship_log_add`：写入单条实习事实记录。
- `format_internship_log_entry`：格式化写入确认。
- `internship_log_list`：按日期、类别或项目查询记录。
- `internship_log_search`：按关键词搜索记录。
- `format_internship_log_list`：格式化查询结果。
- `internship_daily_report_generate`：生成指定日期日报草稿。
- `format_internship_daily_report`：格式化日报草稿。

除非用户明确要求长期偏好或稳定背景沉淀，否则不要使用 `memory_write`。结构化实习事实必须进入 internship 记录工具。
